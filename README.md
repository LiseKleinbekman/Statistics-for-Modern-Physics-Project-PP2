## Aim of the project
The aim of the project is to fit a background curve using a published dijet mass spectrum from HEPData. In addition the goal is to look for a localized excess and compare both models using the likelihood-ratio. 
## Set-up
Clone the repository:\ 
git clone <git@github.com:LiseKleinbekman/Statistics-for-Modern-Physics-Project-PP2.git+>

Go into the repo root:\
cd Statistics-for-Modern-Physics-Project-PP2
### Requirements
Python version: 3.14 + 

Install the following packages in the terminal: \
python -m pip install requirements.txt

Create a virtual enviroment:\
python -m venv venv

Activate it:\
(macOS/Linux)
source venv/bin/activate

(Windows)
venv\Scripts\activate
### Data
The data used is from the following paper: https://arxiv.org/pdf/1910.08447

The data can be downloaded as .csv file using the following link: https://www.hepdata.net/record/93086

In this repository includes the data set downloaded from https://www.hepdata.net/record/93086 and can be found in data/data_.csv
